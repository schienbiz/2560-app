## 什麼是 2560 戰法

利用**快線 MA25**（25 日移動平均）與**慢線 MA60**（60 日移動平均）的相對位置，判斷趨勢方向與進出場時機。

| 訊號 | 條件 | 意義 |
|------|------|------|
| ▲ 黃金交叉 | MA25 由下往上穿越 MA60 | 趨勢轉多，買入訊號 |
| ▼ 死亡交叉 | MA25 由上往下穿越 MA60 | 趨勢轉空，賣出訊號 |
| 無訊號 | 兩線未發生穿越 | 觀望，等待訊號確認 |

均線 = 過去 N 天收盤價的算術平均值，反映中期趨勢方向。

## 自選清單

主頁面。顯示你追蹤的所有標的及最新訊號狀態。

### 新增標的

點擊右上角 **＋ 新增** → 輸入代碼 → 確認新增。

| 資產類型 | 代碼格式 | 範例 |
|---------|---------|------|
| 台股 | 純數字 | `2330`、`0050` |
| 美股 | 英文代碼 | `AAPL`、`TSLA` |
| 加密貨幣 | 交易對 | `BTCUSDT`、`ETHUSDT` |

### 即時價格

新增標的後頁面會自動連線，顯示即時報價與目前 MA 相對位置（`MA25>MA60` 或 `MA25<MA60`）。

### 立即掃描

點擊 **⚡ 掃描** — 對所有自選標的重新計算訊號，顯示：
- 目前訊號（黃金交叉 / 死亡交叉 / 無訊號）
- 即時收盤價與 MA 相對位置
- 高信心度標記

> 點擊 **高信心 ⓘ** 可查看判斷條件說明。

### 設定標的（⚙）

點擊每個標的右側的 **⚙** 按鈕，可調整：

**均線組合**
- 三個常用預設：`5/20`、`25/60`（預設）、`50/200`
- 也可直接在輸入框填入任意數值

**通知**
- 開關黃金交叉 / 死亡交叉通知
- 啟用或暫停此標的的所有通知

**進階設定**（點 `▶ 進階設定` 展開）
- **顯示名稱**：設定自訂標籤，如「台積電」、「BTC」
- **接近快線警示門檻**：價格進入 MA 快線附近時額外通知（預設 1.5%）

**移除此標的**
- 點一次 → 按鈕變為「確認移除？」
- 再點一次 → 確認刪除

## 圖表與分析

在圖表頁輸入代碼並點 **查詢**，或從自選清單點進標的自動帶入。

### 三層圖表

圖表分為三個同步區塊，拖動任一區塊時間軸，其他兩個同步移動：

**① K 線 + 均線**（主圖）
- 近 120 天的 OHLC K 線
- **MA 快線**（藍色）：預設 MA25
- **MA 慢線**（橘色）：預設 MA60
- 擺動結構標記（可切換）：`↑HH` 更高高點、`↓HL` 更高低點、`↑LH` 更低高點、`↓LL` 更低低點

**② RSI(14)**（下方子圖）
- 相對強弱指數，衡量近期漲跌動能強弱
- 水平參考線：70（超買）、50（中性）、30（超賣）
- RSI > 50 且持續上升 → 多方動能強；RSI < 50 且下降 → 空方動能強

**③ MACD(12/26/9)**（最下方子圖）
- 柱狀圖（histogram）：綠色 = 多頭動能，紅色 = 空頭動能
- 藍線 = MACD 線（快速 EMA − 慢速 EMA）
- 橘線 = 訊號線（MACD 的 9 日 EMA）
- 柱狀由負轉正（翻綠）代表動能由弱轉強；由正轉負（翻紅）代表動能由強轉弱

### 訊號橫幅

圖表上方顯示：最新訊號、發生幾天前、信心度，以及目前的 RSI 數值與 MACD 方向（▲ 正向 / ▼ 負向）。

### 訊號信心度

信心度由 **4 個獨立因子**評分，每個因子達標得 1 分：

| 因子 | 達標條件 |
|------|---------|
| 成交量確認 | 當日成交量 > 10 日均量 × 1.2 |
| 價格貼近均線 | 收盤價在 MA 慢線 ±15% 以內 |
| RSI 方向確認 | 黃金交叉時 RSI > 50；死亡交叉時 RSI < 50 |
| MACD 動能確認 | 黃金交叉時 MACD 柱狀 > 0；死亡交叉時柱狀 < 0 |

**評分結果：**
- 3–4 分 → **高信心度**：四個面向共同確認趨勢
- 2 分 → **中信心度**：部分確認，可關注但需謹慎
- 0–1 分 → **低信心度**：訊號出現但動能未充分支撐

### 進退場分析卡

自動計算並顯示：

| 指標 | 說明 |
|------|------|
| 進場建議 | 適合進場 / 可考慮進場 / 追高風險 / 避免做多 |
| MA 距離 | MA 快線與慢線的差距百分比 |
| RSI(14) | 目前 RSI 數值與超買/超賣狀態 |
| MACD 柱狀 | 目前動能方向與數值 |
| 理想進場區 | MA 快線 ±1% 的價格區間 |
| 策略停損 | MA 慢線數值（跌破視為趨勢改變） |

點擊 **▶ 判斷邏輯說明** 展開各狀態的觸發條件。

### AI 分析（多模型交叉校正）

點擊 **✦ AI 分析** — 系統同時向多個 AI 模型（NVIDIA、Groq、OpenRouter）查詢，交叉比對後合成一份更精準的分析。

分析內容涵蓋：
- 均線位置與趨勢判斷
- RSI 動能是否確認方向
- MACD 柱狀趨勢與信號線關係
- 近期支撐 / 壓力觀察
- 進退場建議（具體進場區、停損位）
- 整體操作偏向

> 若各模型出現明顯分歧，分析開頭會標示 **⚠ 分析有分歧：** 並說明不同看法，由使用者自行判斷。

### 記錄交易 / 設定提醒

- 點 **＋ 記錄交易** → 填入進場日期、價格、方向（做多/做空）、備註 → 確認
- 點 **🔔 設定提醒** → 選擇日期 → 填入提醒事項 → 確認

## 回測

在圖表頁點擊 **📊 回測** — 以歷史資料模擬交叉訊號的實際績效。

### 回測邏輯

- 掃描近 1 年（預設）的歷史 K 線
- 每次 MA 快線與慢線發生穿越即視為一筆交易
- 入場價 = 訊號當日收盤價
- 出場 = 下一次反向交叉收盤

### 回測結果說明

| 欄位 | 說明 |
|------|------|
| 已結清 | 已出場的模擬交易筆數 |
| 勝率 | 獲利筆數 ÷ 已結清總筆數 |
| 平均報酬 | 每筆已結清交易的平均漲跌幅 |
| 最佳 / 最差 | 單筆最高獲利 / 最大虧損 |
| 持倉中 | 尚未出場的最後一筆，顯示目前未實現損益 |

每筆交易條目顯示：方向（黃金/死亡）、信心度（4 因子）、持有天數、損益。

> 回測為事後統計，不代表未來績效。滑點與手續費未納入計算。

## 提醒

**提醒清單**：未來的提醒事項，按日期排序。今天到期的以黃色標示。

**提醒歷史**：過去觸發的訊號紀錄，包含：
- 訊號類型（黃金交叉、死亡交叉、接近進場區）
- 發生日期與收盤價
- 5 天 / 10 天 / 20 天後的報酬表現（訊號後 5–20 個交易日陸續填入）

> 「結果計算中」表示訊號剛發生，數據尚在收集。

### 新增提醒

點擊右上角 **＋ 新增** → 輸入標的代碼、日期、備註內容。

## 交易紀錄

記錄每筆根據訊號操作的交易，追蹤實際損益。

### 新增交易

從**圖表頁**點 **＋ 記錄交易**（建議方式，自動帶入代碼與訊號資訊）。

填入欄位：
- **進場日期**
- **進場價格**
- **方向**：做多（Long）或做空（Short）
- **備註**（選填）

### 結清交易

在「持倉中」區塊點擊任一筆交易 → 填入出場日期與出場價格 → 系統自動計算損益 %。

輸入出場價格時，畫面即時顯示預估損益（綠色 = 獲利，紅色 = 虧損）。

### P&L 計算

- **做多**：`(出場價 − 進場價) ÷ 進場價 × 100%`
- **做空**：`(進場價 − 出場價) ÷ 進場價 × 100%`

## 歷史統計

彙整所有**已結清**交易的績效數據。

### 交易統計卡

| 指標 | 說明 |
|------|------|
| 勝率 | 獲利筆數 ÷ 已結清總筆數 |
| 平均報酬 | 每筆已結清交易的平均報酬率 |
| 最大獲利 | 單筆最高獲利 |
| 最大虧損 | 單筆最大虧損（負數） |

數據按訊號類型分組：黃金交叉訊號、死亡交叉訊號、手動記錄。

> 點擊各指標旁的 **ⓘ** 可查看定義說明。

### 訊號績效追蹤

頁面下方自動計算每個標的、每種訊號類型，在訊號觸發後 **5 日 / 10 日 / 20 日** 的歷史表現：

| 欄位 | 說明 |
|------|------|
| 勝率 | 以訊號收盤價為基準，N 日後上漲的比例（黃金交叉）|
| 平均報酬 | N 日後相對訊號當日的平均漲跌幅 |

此數據由系統每日自動更新，累積的訊號越多，統計越可靠。

## 進階：自訂均線組合

預設使用 MA25/MA60（2560 戰法），但任何均線組合都支援。

**常用組合參考：**

| 組合 | 適用場景 |
|------|---------|
| **5/20** | 短線波段、加密貨幣、ETF |
| **25/60** | 台股中線操作（預設） |
| **50/200** | 長線趨勢、美股藍籌 |

修改方式：自選清單 → ⚙ → 點選預設組合 **或** 直接輸入自訂數值 → 儲存。

儲存後，掃描、圖表、回測、通知全部自動使用新的均線組合。

## 通知設定

通知透過你登入的平台（LINE 或 Telegram）推送。

觸發條件（可在各標的設定中個別開關）：
- **黃金交叉**：MA 快線向上穿越慢線
- **死亡交叉**：MA 快線向下穿越慢線
- **接近快線**：價格進入 MA 快線 ±1.5% 區間（門檻可調整）
- **離開進場區**：接近快線後，價格又移動超過 3% 離開，提醒進場窗口已關閉

每日掃描時間：台股收盤後自動執行（約 16:00–16:30 台灣時間）。

> 若想暫停特定標的通知但保留追蹤：⚙ → 通知 → 關閉「啟用通知」。

### 通知格式

**黃金交叉 / 死亡交叉通知：**
```
🟢 台積電 黃金交叉 高信心度
MA25 1050.00 ↑ MA60 980.00 · 收盤 1065
RSI 58.3 · MACD柱 +12.45
情緒 📈 正面 · 極度恐慌(22) — 反向買入訊號（crypto 限定）
進場區 1039–1061，跌破 980.00 停損
[AI 摘要分析一段]
```

**接近快線通知：**
```
📍 BTCUSDT 接近 MA25 進場區
距 MA25 僅 0.87% · 收盤 95200
RSI 52.1 · MACD柱 +185.30
進場區 94248–96152，跌破 89500.00 停損
[AI 摘要分析一段]
```

> **情緒指標**（加密貨幣限定）：使用 Alternative.me 恐懼貪婪指數（0–100）。極度恐慌時黃金交叉屬反向買入訊號；極度貪婪時死亡交叉屬分配警示。台股標的不顯示此欄。

## 常見問題

**Q：為什麼沒收到通知？**
A：確認 ⚙ 設定中「啟用通知」已打開，且 LINE / Telegram 帳號已正確綁定。

**Q：加密貨幣跟台股的資料來源不同嗎？**
A：是。台股使用 Yahoo Finance，加密貨幣使用 Kraken。兩者的日曆天計算方式不同，加密貨幣全年無休，台股有交易日限制。

**Q：為什麼我的 MA200 標的有時沒有訊號？**
A：MA200 需要約 320 個日曆天的歷史資料（台股約 221 個交易日）才能計算。新上市或資料不足的標的會暫時跳過。

**Q：RSI 和 MACD 是什麼，我需要懂嗎？**
A：不需要深入理解，系統已將它們納入信心度評分。簡單原則：RSI 在 50 以上且 MACD 柱狀為正，代表動能支持趨勢；反之則謹慎。通知訊息中的 `RSI · MACD柱` 一行可快速判斷動能狀況。

**Q：AI 分析說「分析有分歧」是什麼意思？**
A：系統同時詢問多個 AI 模型，當不同模型對趨勢方向有不同判斷時，會主動標示出來。這不代表分析失敗，而是提醒你訊號可能處於不確定狀態，需要更謹慎評估。

**Q：回測結果可以直接照做嗎？**
A：回測是事後統計，用於了解策略在歷史上的勝率與特性，不是預測。實際交易還需考量滑點、手續費、持有成本與個人風險承受度。

**Q：訊號後的 5/10/20 天報酬是怎麼算的？**
A：以訊號當天收盤價為基準，分別計算之後第 5、10、20 個交易日的漲跌幅。這是事後統計，不是預測。

**Q：可以同時追蹤台股和加密貨幣嗎？**
A：可以。自選清單支援混合標的，每個標的獨立設定均線組合與通知條件。
